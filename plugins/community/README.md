# Community Plugins

This folder is for user-generated BetterTouchTool plugins that have been reviewed
and accepted into the public repository.

Acceptance into this folder means:

- the plugin has a clear description and reviewable source code
- the plugin follows the folder and metadata conventions from `plugins/_template`
- obvious safety, privacy, and reliability issues were checked during review
- the plugin is suitable to be shown in a future BetterTouchTool plugin gallery

It does not mean the plugin is endorsed for every workflow or audited like a
security-critical dependency. Native Swift plugins run inside BetterTouchTool's
process and should be reviewed with care.

New submissions should start as a folder copied from `plugins/_template`.
