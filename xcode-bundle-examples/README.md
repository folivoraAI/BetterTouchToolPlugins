# Xcode Bundle Examples

This folder contains the older Xcode-based plugin development project.

Use it when a plugin needs:

- multiple source files
- bundled resources
- a custom framework setup
- third-party dependencies
- explicit signing and notarization workflows

For most new plugins, prefer the Swift source plugin folders in `../plugins`.

Open `BetterTouchToolPluginDevelopment.xcodeproj` in Xcode and build the sample
target you want to inspect.
